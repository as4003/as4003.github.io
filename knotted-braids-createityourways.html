KNOTTED - BRAIDS:
CREATE IT YOUR WAYS

This is about how to make a knotted braid at home out of a group of strings.

1. Cut 6 (or however many you want) long, different, coloured strings of the same size.

2. Tie the strings together at the top of your pattern.

3. Line each string up on different sides.

4. Label each string with different numbers.

5. Pick up the left string and make a loop.

6. Loop the left string below the rest of the strings while making a gap.

7. Then, loop the string on top of the other strings.

8. Insert the left string back into the gap to knot all the strings.

9. Repeat 5, 6, 7 and 8 for the right string, then the center right, center left and the 2 in the center.

10. Continue this process until making the knotted string patterns.
